# Documents Directory

Place your PDF files here to make them accessible via the `/api/documents/[filename]` route.

## Whitepaper

To make the whitepaper available, place your PDF file here with one of these names:

- `CRYPTORAFTS WHITE PAPER .pdf` (exact filename)
- `whitepaper.pdf` (simplified name)
- `CRYPTORAFTS-WHITE-PAPER.pdf` (alternative)

The API will automatically find and serve the file when accessed via:
- `/api/documents/CRYPTORAFTS%20WHITE%20PAPER%20.pdf`
- `/api/documents/whitepaper.pdf`

## Other Documents

You can also place other PDF documents here and access them via the API route.
